# BenEl Levy - Research Portfolio Landing Page

Static GitHub Pages site generated from the research portfolio PDF.

## Deploy on GitHub Pages

1. Copy these files into your GitHub Pages repository.
2. Commit and push:
   ```bash
   git add .
   git commit -m "Build research portfolio landing page"
   git push
   ```
3. In GitHub, go to **Settings -> Pages** and publish from the `main` branch.

## Files

- `index.html` - landing page
- `styles.css` - visual design
- `assets/BenEl_Levy_Research_Portfolio.pdf` - downloadable original portfolio
- `assets/page-*.jpg` - visual previews rendered from the PDF
